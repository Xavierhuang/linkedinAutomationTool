## Troubleshooting

### "I don't see the Materials button"

**Solution 1: Hard Refresh**
```
Windows: Ctrl + Shift + R
Mac: Cmd + Shift + R
```

**Solution 2: Clear Cache**
```
1. Open browser DevTools (F12)
2. Right-click refresh button
3. Select "Empty Cache and Hard Reload"
```

**Solution 3: Check Console**
```
1. Open browser DevTools (F12)
2. Go to Console tab
3. Look for any errors
4. Share errors if you see any
```

### "The button doesn't work"

**Check:**
1. Do you have an API key configured? (Settings > API Providers)
2. Is the backend running?
3. Check browser console for errors

## Need Help?

If you still can't find the button:
1. Take a screenshot of your Organizations tab
2. Check browser console for errors
3. Verify you're on the latest version
4. Try a hard refresh (Ctrl+Shift+R)
